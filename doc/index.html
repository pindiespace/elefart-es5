public/js/elefart-controller.js

elefart.controller The Controller module for the application. Traps user input, also routes in-game events to other modules (e.g. elefart.display). Requires the wrapper event methods in elefart.dom. Supports:

  • standard mouse desktop events
  • mouse events on touch devices
  • keyboard events
  • virtual keyboard events

public/js/elefart-netmessage.js

CommonJS object available on client and on server common file between client and server provding a dictionary for decoding messages between user programs each message may be associated with additional data, shown to the right of the message

NOTE: the function is 'wrapped' in an 'exports' object whose configuration is compatible with what NodeJS needs, and can also be used by your client. On the browser, the 'common' object is created. On the server, it already exists. Format follows CommonJS format.
